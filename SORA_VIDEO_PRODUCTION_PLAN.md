# Vue Roadmap Video Production Plan

## Best Approach
For this app, the strongest video is a hybrid:
- use a real screen recording for the exact app walkthrough
- use ChatGPT and Sora for the intro, outro, and short transition visuals

This gives you:
- accurate app screens
- better trust for viewers
- polished branded moments without needing full manual editing

## Why This Is The Best Choice
Sora is great for generating video creatively, but a real app walkthrough needs exact UI fidelity. A generated video may not reproduce the app interface accurately enough for a tutorial.

So:
- `record the app screen` for anything instructional
- `use Sora` for branded opening and closing clips

## Video Structure
### 1. Intro clip
Use Sora for a short branded intro with the `Vue Roadmap` feeling.

### 2. Screen recording walkthrough
Record the app while following the walkthrough script.

### 3. Outro clip
Use Sora for a closing motivational clip.

## Scene Plan
### Scene 1: Intro
- Type: Sora
- Length: `5 to 8 seconds`
- Goal: introduce the brand and learning mood

### Scene 2: Install
- Type: real screen recording
- Goal: show the APK being opened and installed

### Scene 3: Open app and start sprint
- Type: real screen recording
- Goal: show `Today` and `Start Sprint Today`

### Scene 4: Explain Today
- Type: real screen recording
- Goal: show objective, study topics, research prompts, build tasks, and comfort note

### Scene 5: Explain Roadmap
- Type: real screen recording
- Goal: show full 14-day structure

### Scene 6: Explain Checklist
- Type: real screen recording
- Goal: show daily progress marking

### Scene 7: Explain Guide
- Type: real screen recording
- Goal: show how users get help inside the app

### Scene 8: Explain Tracker
- Type: real screen recording
- Goal: show confidence scores and notes

### Scene 9: Outro
- Type: Sora
- Goal: end with momentum and encouragement

## Sora Prompt Ideas
### Intro Prompt
Create a warm, inspiring mobile learning promo video for an app called Vue Roadmap. Use a light gold and cream color palette with modern green accents. Show a confident learner starting a 14-day coding journey, abstract mobile learning visuals, progress cards, and uplifting motion graphics. Style should feel premium, clean, hopeful, and educational. Add subtle text moments: "Vue Roadmap", "14-Day Vue Sprint", "Build Daily. Learn Clearly."

### Outro Prompt
Create a short motivational closing video for a learning app called Vue Roadmap. Show progress, confidence, momentum, and growth in a bright light visual style with gold, cream, and green colors. End with a polished title card that says "Start Today" and "Build Daily. Learn Clearly."

### Transition Prompt
Create a short transition animation for a mobile coding study app. Use soft light gradients, premium UI-inspired motion, and a clean educational feel. Colors should match a warm cream, gold, and green app theme.

## Recording Checklist
- Use the signed release APK
- Record in portrait mode
- Keep taps slow and clear
- Pause briefly before changing tabs
- Follow `WALKTHROUGH_SCRIPT.md`
- Use `walkthrough-captions.srt` after editing

## Editing Order
1. Generate intro and outro in Sora
2. Record the app screen locally
3. Put the clips in order
4. Add voice using the walkthrough script
5. Add subtitles using the `.srt` file
6. Export a vertical video for phone sharing

## Recommended Final Exports
- `1080 x 1920` for vertical sharing
- `720 x 1280` if you want a smaller file
