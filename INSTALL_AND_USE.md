# Install And Use Vue Roadmap

## Install The App
1. Get the file `app-release.apk`.
2. Open it on your Android phone.
3. If Android asks, allow install from that source.
4. Finish the installation and open `Vue Roadmap`.

## First Time Use
1. Open the `Today` tab.
2. Tap `Start Sprint Today`.
3. Read the objective and study guidance for the current day.

## What Each Tab Does
- `Today`: tells you what to learn right now
- `Roadmap`: shows the full 14-day plan
- `Checklist`: lets you mark your daily progress
- `Guide`: explains how to use the app
- `Tracker`: stores your confidence and notes for each day

## Best Daily Routine
1. Read the `Today` page first.
2. Study the listed topics.
3. Build the day task from scratch.
4. Mark progress in `Checklist`.
5. Record how the day felt in `Tracker`.

## If You Feel Stuck
- Read the day's objective again.
- Focus on `What you need to know`.
- Use the `What to study` and `What to research` sections.
- Write your confusion into `Tracker` instead of ignoring it.

## Important Note
Progress is saved locally on the phone. If you reset the sprint, saved progress and notes are cleared.
