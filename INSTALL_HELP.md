# Install Help

If `Vue Roadmap` does not install on a phone, check these first:

1. Uninstall any older `Vue Roadmap` app already on the phone.
2. Use the latest signed release APK, not a debug APK.
3. Open the APK from the phone's `Files` or `Downloads` app after it finishes downloading.
4. Allow installs from that source if Android asks.
5. Confirm the phone is running Android `6.0` or newer.
6. Make sure the phone has enough free storage.

## Common reasons installation fails

- An older build signed with a different key is already installed.
- The phone blocks sideloaded apps from chat apps like WhatsApp.
- The Android version is too old for this app.
- The APK was interrupted or corrupted while downloading or forwarding.

## Best sharing methods

- Google Drive
- Dropbox
- GitHub Releases
- Direct website download

These usually work better than forwarding the APK through WhatsApp.
