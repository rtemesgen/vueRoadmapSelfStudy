# Vue Roadmap Walkthrough Script

## Video Goal
Show a new user how to install the app, start the sprint, understand the tabs, and use it daily with confidence.

## Recommended Length
`3 to 5 minutes`

## Voiceover Script

### Scene 1: Intro
Hello, this is Vue Roadmap, a mobile app designed to guide you through a 14-day Vue 3 self-study sprint.

### Scene 2: Install
After downloading the release APK, install the app on your Android phone and open it.

### Scene 3: Today Tab
When the app opens, start in the Today tab. Tap Start Sprint Today. This becomes your main learning screen for the whole sprint.

### Scene 4: Daily Guidance
Each day shows your objective, what you need to know, what to study, what to research, what to build, and how to know you are on track.

### Scene 5: Roadmap Tab
If you want to see the full journey, open the Roadmap tab. It contains all 14 days with detailed guidance so you always know where you are heading.

### Scene 6: Checklist Tab
Use Checklist to mark your daily routine, checkpoint progress, and completed days. This helps you stay consistent and see your momentum.

### Scene 7: Guide Tab
If you forget how the app works, open the Guide tab. It explains what each screen does and how to move through the sprint.

### Scene 8: Tracker Tab
Use Tracker to record your confidence for each day and save short notes about what felt easy or difficult. This makes review much easier later.

### Scene 9: Mini Project Direction
As you move through the sprint, the app also points you toward the mini project you are building so your practice stays connected to a real goal.

### Scene 10: Closing
The best way to use Vue Roadmap is simple: open Today, follow the guidance, build the feature, check your progress, and write a short note before you stop.
