# Vue Roadmap Self Study

This repository contains the `Vue Roadmap` mobile app in:

```text
roadmap-app/
```

## What It Is
- A small `Ionic Vue + Capacitor` mobile app
- Built to guide a 14-day Vue 3 self-study sprint
- Includes daily objectives, study topics, research prompts, build tasks, checklists, and progress tracking

## Main App Folder
- [roadmap-app](c:\Users\HAVEN\Desktop\kgl ppt\roadmap-app)

## What Users Should Do
1. Install the release APK.
2. Open the app and go to `Today`.
3. Tap `Start Sprint Today`.
4. Follow the objective, study topics, research prompts, and build steps for the day.
5. Use `Checklist` and `Tracker` to save progress.

## App Tabs
- `Today`: the guided lesson for the current day
- `Roadmap`: the full 14-day plan with details for every day
- `Checklist`: daily routine and completion tracking
- `Guide`: how to use the app clearly
- `Tracker`: confidence scores and daily notes

## Local Development
```bash
cd roadmap-app
npm install
npm run dev
```
