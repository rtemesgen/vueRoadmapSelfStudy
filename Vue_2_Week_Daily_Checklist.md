# Vue 3 Daily Checklist

Use this page every day. Check items only when you actually did the work.

## Daily Workflow Template
- [ ] Spend `45-60 min` studying the day's concept
- [ ] Build the day's feature from scratch
- [ ] Debug one issue without guessing blindly
- [ ] Refactor at least one small piece of code
- [ ] Write a short recap from memory
- [ ] Explain the feature out loud in simple language
